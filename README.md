## Getting started
```bash
hugo new site YOURBLOG
cd YOURBLOG
git clone https://github.com/Stradichenko/krota-template themes/krota-template
echo "theme = 'krota-template'" >> config.toml
cp themes/krota-template/style.css static/
```
